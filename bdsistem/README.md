# Base de datos SEIM

Esta carpeta contiene la base de datos lista para importar en MySQL.

## Archivos
- `schema.sql`: esquema completo actualizado (incluye `module_records`).
- `seed.sql`: datos de prueba.
- `full.sql`: esquema + datos para cargar directamente en el hosting MySQL.
